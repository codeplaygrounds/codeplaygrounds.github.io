About Code Playgrounds

Code Playground is a online editor of programming language. It is designed to make the programming life easy. Here programmer/learning can write, there program and learn that. It’s supports for multiple programming languages like PHP, HTML, CSS, Java Script, jQuery, Bootstrap and Angular. Here user can easily add the external resources by just one click and can see the live preview of their codes.

Features

  • Easy to write, easy to learn and easy to share.
  • Real-time code collaboration.
  • Fully-featured, syntax and line editor.
  • Run time preview of code changes.
  • Easy to share, either codes or results (output of code).
  • Explore coding style of other programmers.
  • Explore, share and download your’s and others codes to learn.
  • Create multiple file as your requirements.
  • Support for PHP, HTML, CSS, Java Script, jQuery, Bootstrap and Angular.
  • Support for adding external resource.
  • And much more to come…
